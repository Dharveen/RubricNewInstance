# Rubric Website Redesign

## Project Overview
This repository contains the end-to-end redesign of the Rubric website. Rubric is a technology consulting and software quality engineering company. The goal of this project is to create a modern, premium, and professional digital presence that aligns with the new Rubric brand identity.

## Mission Statement
To represent Rubric as a leader in software quality and technology transformation through a structured and impactful web experience.

## Branding
The redesign utilizes the "Rubric" brand identity, focusing on structure, transformation, and measurable impact.
